# 🎨 Kiro Paso a Paso — Modo Remix

Usa la estructura del workshop como plantilla y adáptala a otro dominio con Kiro.

## Cómo empezar

1. Abre esta carpeta como workspace en Kiro (Autopilot)
2. En el chat, escribe `#workshop-guide` y dile qué dominio quieres:
   - "Quiero hacer un conversor de monedas"
   - "Adapta esto a un conversor de peso"
   - "Hazme un conversor de unidades de datos"
3. Kiro te guía por las 9 sesiones adaptando todo a tu dominio

## Ideas de remix

| Dominio | Unidades ejemplo |
|---------|-----------------|
| 💰 Monedas | USD, EUR, COP, MXN, ARS |
| ⚖️ Peso | kg, lb, oz, g |
| 📏 Distancia | km, mi, m, ft |
| 💾 Datos | GB, MB, KB, TB |
| ⏱️ Tiempo | horas, min, seg, días |
